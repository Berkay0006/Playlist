                                                       Playlist

This project is a C program that manages a playlist using a singly linked list. The program reads track titles from a text file, stores each track as a node in the linked list, and prints the playlist in the terminal. It then removes one track from the playlist and inserts a new track at a chosen position. After the playlist has been updated, the program saves the new playlist to an output file. This shows how linked lists can be used to load, edit, display, and save data in a simple playlist system.
